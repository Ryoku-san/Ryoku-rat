# Ryoku RAT (Educational Project)

A simple remote access tool (RAT) built with Python.  
Used to demonstrate basic client-server communication over LAN.

## Features
- Remote shell communication (basic)
- Screenshot feature (via pyautogui)
- Client compiled to .exe with PyInstaller

## How to Use
1. Run `server.py` on your main PC
2. Edit `client.py` and put your server's IP
3. Compile `client.py` with PyInstaller (optional)
4. Run client.exe on another PC (same Wi-Fi)
5. Server receives and interacts with client
